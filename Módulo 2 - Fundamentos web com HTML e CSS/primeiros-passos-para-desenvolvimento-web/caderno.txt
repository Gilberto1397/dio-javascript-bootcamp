-Como funciona a internet
    -Redes: Conexão entre computadores
    -BackBone: Responsável por fazer o trabalho de conexão entre diversas máquinas
    -Provedor de acesso: Empresas que provém o acesso ao BackBone (internet)    / normalmente empresas de telefone
    -Provedor de serviço: O provedor fornece os diversos possíveis serviços como: (Dial-up(Discada), ADSL(banda larga), Fibra ótica, Rádio, Satélite, Móvel. P2P(peer to peer))

    Caminho da internet até o PC: www > DNS(Server Name Dominion) > IP > Classes de redes (A,B,C,D,E) > 127.0.0.1(localhost)

    Teste de ping à um site no terminal -   ping Site

-TCP/IP, portas, roteadores, switches e modems
    -TCP/IP e UDP
        TCP/IP: Protocolos de comuicação entre computadores em rede    
            Transmission Control Protocol
        IP: Internet protocol - Protocolo de internet

        4 Camadas para estabelecer o TCP/IP
            -Física: Placa de rede...
            -Rede: Conexão
            Transporte: TCP, UDP...
            Aplicação: FTP, SMTP, HTTP

            UDP: Rápido, Não confiável, Qualquer um pode receber sem controle disso, Uso em livestreams, pois é uma conexão que se preocupa mais com o enviar dos dados, essa é a prioridade
            TCP: Voltado a conexão, Maior controle das informações transitadas entre os dispositivos, Serve para manter a integridade da ordem dos dados enviados, Exemplo de uso em apps de mensagens

    -Portas 
        Requisições:
        -FTP: Apenas para envio de arquivos, normalmente porta 20
        -SSH: Exemplo de conexão segura, normalmente porta 22
        -SMTP: Envio de email, normalmente porta 25
        -DNS: Traduz o ip para dominio e vice versa, normalmente porta 53
        -HTTP: Requisição simples na net, normalmente porta 80
        -HTTPS, Requisição segura, normalmente porta 443

    -Roteadores, switches e modems
        -Modem: Converte os dados da rede para serem interpretados pelo PC e vice versa
        -Roteador: Envia os dados para diversos dispositivos - envia para todos sem distinção de quem pediu ou não
        -Switch: Mesmo q roteador mas só entrega o que foi pedido em cada dispositivo
